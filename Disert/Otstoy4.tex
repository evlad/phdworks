По материалам дипломной работы + описание инструментальной
программной системы.

Излагается подход, используемый автором для программирования
задач, связанных с моделированием САУ и обучением нейронных сетей
для задач предсказания временных рядов, идентификации САУ и
синтеза регулирующих устройств.  В основе подхода - вычислительная
сеть Петри с необходимыми расширениями.

Важные отличительные черты подхода:
\begin{itemize}
  \item Естественное отображение модульной структуры САУ на программные
  модули.
  \item Высокий уровень абстракции при программировании отдельных
  функциональных модулей.
  \item Декларативная форма задания связей между модулями в
  требуемую схему.
  \item Возможность включения в контур моделируемой САУ объектов
  реального мира.
  \item \ldots
\end{itemize}
