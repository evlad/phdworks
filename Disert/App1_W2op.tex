% Объект управления второго порядка с физически нереализуемым
% Винеровским регулятором
\label{W2op_case}

В данной серии экспериментов управляемый объект представляет собой
дискретный аналог колебательного звена второго порядка с затуханием.
Рассмотренный вариант объекта управления имеет две особенности.
Во-первых, целесообразно исследовать поведение нейросетевого
регулятора при управлении объектом, обладающим колебательными
свойствами.  Во-вторых, при выбранных спектральных плотностях уставки
и помехи оказывается, что оптимальный Винеровский регулятор физически
нереализуем, поэтому представляет интерес результат синтеза НОР в
данном случае.

Приводимые для случая $C^*_{WOC_2}(z)$ результаты соответствуют замене
замкнутого контура с $P^*_2(z)$ оптимальным Винеровским фильтром.  Эти
результаты не имеют физического смысла и приводятся только для
ориентира при сравнения ПИД и НОР.  Очевидно, что эксперимент $E_p$
невозможно провести для $C^*_{WOC_2}(z)$ с помощью описанной замены,
так как параметры объекта управления отличаются от номинального.

\subsection{Номинальные условия}%
\label{W2op_nominal_cond}

\begin{itemize}

\item
Формирующий фильтр уставки:
$R^*_2(z)=\mathcal{Z}\Bigl\{\displaystyle\frac{6}{1+6s}\Bigr\}
         =\displaystyle\frac{z}{z-0.8465}$

\item
Формирующий фильтр помехи:
$N^*_2(z)=0.05$

\item
Винеровский оптимальный фильтр:
$W^*_2(z)=\displaystyle\frac{0.9975z}{z-0.0021}$

Среднеквадратичная ошибка управления $\MSE_{min}=0.0025$

\item
Объект управления:
$$P^*_2(z)=\mathcal{Z}\Bigl\{\displaystyle\frac{0.6}{9.09+6s+s^2}\Bigr\}
          =\displaystyle\frac{0.029z}{z^2-0.095z+0.0025}$$

\item
Винеровский оптимальный регулятор (физически нереализуем):
$$C^*_{WOC_2}=\displaystyle\frac{33.8988z^2-3.225z+0.084}{z-1}$$

\item
Исходный ПИ регулятор (см. \figref{fig:dw2op_pid_step}):
$
C^*_{PI_2}(z)=12 +
              12\displaystyle\frac{z}{z-1}
$

\end{itemize}

Архитектура НС--О: $\NN^o_{1+1,1}$

\item
Коэффициенты скорости обучения: $\eta_h=0.0001$, $\eta_o=0.00001$

\item
Обучающая выборка: длина $L=200$, диапазоны амплитуд сигналов:
$u=[-109.87, 0.177.66]$, $y=[-3.43, 5.57]$
%$u=[-109.8710, 0.177.6638]$, $y=[-3.4312, 5.5725]$

\item
Контрольная выборка: длина $L=200$, диапазоны амплитуд сигналов:

%$u=[-94.4904, 150.8581]$, $y=[-2.9317, 4.5471]$

\item
Обучение в течение 95 эпох до начала роста $\MSE$ на контрольной выборке.

\end{itemize}

\subsection{Параметры дообучения НС--Р в контуре}%
\label{W2op_NNCfin_descr}

\begin{itemize}

\item
Коэффициенты скорости обучения: $\eta_h=0.00001$, $\eta_o=0.000005$

\item
Длительность эпохи $L=500$

\item
Проверка гипотезы о постоянстве среднего значения $\MSE$
осуществлялась на базе 50 эпох.  Для проверки использовался
приближенный критерий серий, основанный на медиане с уровнем
значимости критерия $\alpha=5\%$.

\item
Гипотеза постоянства среднего значения $\MSE$ оказалась истинной на
второй по счету базе, то есть, обучение было остановлено после 100
обучающих эпох.

\end{itemize}

\subsection{Неноминальные условия}%
\label{W2op_differ_cond}

\begin{description}

\item[$E_{r-}$:]
Уменьшенная мощность уставки: $R^*_2(z)=\displaystyle\frac{0.5z}{z-0.8465}$

\item[$E_{r+}$:]
Увеличенная мощность уставки: $R^*_2(z)=\displaystyle\frac{2z}{z-0.8465}$

\item[$E_{n-}$:]
Уменьшенная мощность помехи: $N^*_2(z)=0.025$

\item[$E_{n+}$:]
Увеличенная мощность помехи: $N^*_2(z)=0.1$

\item[$E_m$:]
Уставка представляет собой последовательность прямоугольных импульсов
амплитудой 1, периодом 40 и длительностью импульса 20 дискретных
отсчетов времени.  Общая длина временного ряда составила 150
дискретных отсчетов времени.

\item[$E_s$:]
Уставка представляет собой гармонический сигнал амплитуды 1 и заданной
частоты.  Для исследования качества управления на различных частотах
были проведены эксперименты с периодом гармонического сигнала от 2 до
50 дискретных отсчетов времени.  Общая длина временного ряда была
равна 500 дискретным отсчетам времени.  Эксперименты проводились с
нулевой помехой для того, чтобы выяснить идеальный вид зависимости
качества управления от частоты.

\item[$E_p$:]
Объект управления второго порядка с другими параметрами:
$$P^*_{2'}(z)=\displaystyle\frac{0.06z}{z^2-0.095z+0.005}$$

\end{description}
